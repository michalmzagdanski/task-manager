# Task Manager


Task Manager is an app where users can add, edit, delete, sort, and filter tasks. Tasks are saved in LocalStorage, so data persists between sessions.

## Features

- Add, edit, and delete tasks
- Search, filter, and sort tasks
- View task details inline below the selected task
- Save tasks with LocalStorage persistence

## Technologies

- HTML
- JavaScript
- CSS

## Installation

1. Download the project files.
2. Open the project folder.
3. Open index.html in your browser.
